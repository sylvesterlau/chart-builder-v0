# Chart builder v0

This is a Figma plugin used for experimentation and quick iteration. Create semi-donut chart with legend.

Quick build

1. Extract the folder
2. Install and build:

```bash
npm install
npm run build
```

3. Use the generated `manifest.json` to import the plugin into Figma.
4. Replace the key in `src/config.ts`

- Theme collection key
- Legend comp key
